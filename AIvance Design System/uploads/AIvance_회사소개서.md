# AIvance 회사소개서

> 본 문서는 `aivance.kr` 및 제품별 서브도메인(`cura.aivance.kr`, `copia.aivance.kr`, `creo.aivance.kr`) 홈페이지 제작을 위한 참고 자료입니다.

---

## 1. 회사 개요

**AIvance** (AI + Advance)는 AI로 업무 효율성을 높이는 실전형 소프트웨어 솔루션을 개발하는 회사입니다.

CodeIgniter 4(PHP) 기반의 검증된 백엔드 아키텍처 위에, 실제 업무에 바로 투입 가능한 AI 자동화 기능을 얹어 — 성형·의료 광고, 이커머스, 기업 웹사이트 — 세 가지 영역의 실무 솔루션을 제공합니다.

### 핵심 철학

| 원칙 | 내용 |
|------|------|
| **실전 코드, 데모 아님** | 모든 제품은 PHPUnit 테스트, PHPStan 정적 분석, GitHub Actions CI/CD, SSH 자동 배포까지 갖춘 프로덕션 레벨 코드베이스 |
| **AI는 도구, 핵심은 업무 자동화** | 화려한 챗봇이 아니라 실제 반복 업무(리포트 작성, 상품 설명, 문의 분류, 이상 탐지)를 대신 처리 |
| **비용 최적화형 AI 설계** | LLM 호출이 꼭 필요한 곳에만 사용하고, 나머지는 휴리스틱(규칙 기반)으로 처리해 비용·지연 최소화 |
| **빠른 납품, 유지보수 가능한 구조** | 코어 재사용 + 테마(껍데기) 교체 방식으로 신규 고객사 적용 기간 단축 |

### 기술 리더십

- CodeIgniter 4 / PHP 최신 버전(8.5) 기반 25년 경력 백엔드 개발
- MSA(마이크로서비스 아키텍처) 기반 API 설계 전문
- AWS · Azure · GCP 멀티 클라우드 아키텍처 설계 및 구현 역량
- JWT / OAuth2(Google·Naver·Kakao) 인증, Swagger/OpenAPI 문서 자동화
- Ubuntu 서버 운영, GitHub Actions 기반 CI(PHPStan + PHPUnit)/CD(무중단 배포) 파이프라인 구축

## 3. 제품 상세

### 3-1. AICura — AI 기반 성형·토탈 광고 솔루션

**cura.aivance.kr**

병원(광고주)과 대행사가 광고 성과를 관리하고, 매출·소진 현황을 매일 AI가 요약해주는 광고 관리 솔루션입니다. 앱/웹 클라이언트를 위한 REST API와 관리자 웹 콘솔을 함께 제공합니다.

**핵심 기능**

| 영역 | 내용 |
|------|------|
| AI 일일 리포트 | 매출 현황·소진(재충전 필요) 현황을 매일 자동 생성 — 자연어 요약 리포트로 제공 |
| 스코프별 리포트 분리 | 운영자(전체) · 광고주(자기 병원) · 대행사(소속 병원 합산) — 각자 자신의 범위만 열람 |
| REST API | JWT 인증 기반, Swagger UI로 API 문서 자동 제공 |
| 앱 로그 분석 | 앱 사용자 행동(이벤트 조회·상담 신청·좋아요 등) 실시간 수집 → 시간별 집계 → 관리자 대시보드 시각화 |
| AI 공급자 교체 가능 | 인터페이스 추상화로 Groq 등 AI 공급자를 코드 수정 없이 교체 가능 |
| 안전한 콘텐츠 렌더링 | AI 생성 리포트를 서버에서 안전하게 HTML 변환(XSS 방어) |

**기술 스택**: PHP 8.4+ · CodeIgniter 4 · FrankenPHP · MySQL 8 · JWT 인증 · Swagger/OpenAPI · Groq AI · PHPStan(level 6) · PHPUnit

**신뢰성**: GitHub Actions CI(정적 분석 + 테스트 자동 실행) → main 브랜치 배포 시 SSH 자동 배포, 무중단 OPcache 리로드

### 3-2. AICopia — AI로 업무 효율성을 높이는 쇼핑몰 솔루션

**copia.aivance.kr**

상품·주문·결제·회원 등급까지 쇼핑몰 운영에 필요한 전 기능을 갖추고, 반복적인 운영 업무를 AI가 대신 처리하는 이커머스 솔루션입니다.

**쇼핑몰 기본 기능**

| 영역 | 내용 |
|------|------|
| 상품 | 카테고리·옵션(SKU)·다중 이미지·재고 관리, 검색·필터·정렬 |
| 주문/결제 | 국내 PG 7종(토스페이먼츠·이니시스·나이스페이·카카오페이·네이버페이·PAYCO) + 무통장입금 |
| 마이페이지 | 주문 내역, 취소/반품/교환, 간편 재주문, 배송지 관리, 쿠폰·포인트 |
| 회원 | 이메일 인증 가입 + 소셜 로그인(Google·Naver·Kakao), 구매 실적 기반 자동 등급제 |
| 관리자 | 대시보드, 상품/주문/재고/배송/쿠폰/포인트/회원등급/매입처/기획전 관리, 엑셀 다운로드 |

**AI 자동화 기능**

| 대상 | 기능 |
|------|------|
| 운영자 | 상품 카테고리 자동 추천, 상품 설명 자동 생성, **이미지 업로드만으로 상품 정보 자동 작성**(Vision), 문의 자동 분류 + 답변 초안, 리뷰 요약 + 부정 리뷰 자동 감지, 매출 AI 분석 리포트, 발주 제안, 이상 주문 탐지 |
| 고객 | 개인화 상품 추천, 시맨틱 검색(오타 교정·의미 확장) |

AI 기능은 Groq · Claude · OpenRouter 중 선택 가능하며, API 키 미설정 시 자동 비활성화되어 기본 쇼핑몰 기능에는 영향이 없습니다. 무거운 AI 작업은 비동기 큐로 처리해 페이지 응답 속도에 영향을 주지 않습니다.

**기술 스택**: CodeIgniter 4(PHP 8.1+) · MySQL/MariaDB · AG Grid(관리자 그리드) · TinyMCE 6 · PhpSpreadsheet · OAuth2 소셜 로그인 · PG 어댑터 패턴(7종)

### 3-3. AICreo — 빠른 납품형 기업 홈페이지·CMS 솔루션

**creo.aivance.kr**

기업 홈페이지 제작을 반복할 때 코어는 재사용하고 테마(디자인)만 교체하는 방식으로, 단순 홈페이지는 3~5일, 중형 사이트는 1~2주 납품을 목표로 설계된 솔루션입니다.

**핵심 기능**

| 영역 | 내용 |
|------|------|
| 동적 페이지 | 슬러그 기반 자동 라우팅, 히어로·서비스소개·공지·CTA 섹션 |
| 게시판 | 다중 게시판, 비회원 게시(비밀번호), 위지윅 에디터, 파일 첨부, 비밀글/공지글 |
| 회원 | 회원가입/로그인, 반응형 GNB(DB 메뉴 기반) |
| SEO | OG 태그·메타 설명·GA·네이버 웹마스터 자동 삽입 |
| 관리자 | 대시보드, 페이지/게시판/메뉴/미디어/문의 관리, 사이트 설정 |
| 테마 시스템 | ZIP 업로드 설치, 클릭 한 번으로 테마 전환, 다수의 샘플 테마 제공 |

**기술 스택**: CodeIgniter 4 · Bootstrap 5 · TinyMCE 6 · MySQL/MariaDB

## 4. 공통 기술 기반 (엔지니어링 신뢰성)

세 제품 모두 동일한 엔지니어링 기준을 따릅니다.

- **정적 분석**: PHPStan level 6 이상 (`composer analyse`)
- **테스트**: PHPUnit 단위·통합 테스트 (`composer test`)
- **CI**: GitHub Actions — push/PR마다 PHPStan + PHPUnit 자동 실행, MySQL 서비스 컨테이너 기반 통합 테스트
- **CD**: main 브랜치 반영 시 SSH 기반 자동 배포, 마이그레이션 실패 감지 시 배포 중단, OPcache 무중단 리로드
- **보안**: 파일 업로드 확장자 화이트리스트, 랜덤 파일명, CSRF 필터, Query Builder(SQL Injection 방지), 뷰 출력 이스케이프(XSS 방지)
- **인증**: JWT(API) / 세션(관리자) 이원화, OAuth2 소셜 로그인 지원
- **문서화**: apiDoc 스타일 규칙에 따른 API 문서화 및 Swagger/OpenAPI 자동 생성
