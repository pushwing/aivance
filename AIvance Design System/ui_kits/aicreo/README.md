# AICreo — Corporate Site UI Kit

빠른 납품형 기업 홈페이지 템플릿의 마케팅 홈을 재현합니다.

- **index.html** — 진입점.
- **app.jsx** — Nav · Hero(통계) · 서비스 그리드 · 공지사항 리스트 · 문의 CTA(폼) · Footer.

**Interactions**: 문의 폼 제출 → 성공 Alert.

**Components used**: Button, Badge, Tag, Card, Input, Alert. 브랜드 액센트는 `creo`(rose).

원본: [pushwing/AICreo](https://github.com/pushwing/AICreo).
