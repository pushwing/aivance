# AICura — Ad Dashboard UI Kit

성형·의료 광고 관리 솔루션의 운영자 대시보드를 재현합니다. (회사소개서 기반 재구성 — 공개 프론트 코드 없음)

- **index.html** — 진입점.
- **app.jsx** — 다크 사이드바 · 스코프 선택 탑바 · KPI StatCard · 주간 매출 차트 · AI 일일 리포트 카드 · 병원별 소진 리스트.

**Components used**: Button, IconButton, Badge, Card, StatCard, Select, Alert. 브랜드 액센트는 `cura`(teal).

원본 컨텍스트: `uploads/AIvance_회사소개서.md` §3-1 (REST API · JWT · AI 일일 리포트 · 스코프별 리포트).
