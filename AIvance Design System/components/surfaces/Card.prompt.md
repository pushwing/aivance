**Card** — the base surface container for grouped content.

```jsx
<Card header="AI 일일 리포트" footer={<Button size="sm">전체 보기</Button>}>
  오늘 매출은 전일 대비 12% 증가했습니다.
</Card>
<Card interactive accent="copia">클릭 가능한 카드</Card>
```

`interactive` adds the hover lift; `accent` adds a product-colored top bar.
