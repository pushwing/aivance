**Button** — the primary tappable action; use for form submits, CTAs, and toolbar actions across all AIvance products.

```jsx
<Button variant="primary" iconLeft="arrow-right">시작하기</Button>
<Button variant="outline" accent="copia">장바구니</Button>
<Button variant="subtle" size="sm" pill>전체</Button>
```

Variants: `primary` (solid brand), `outline`, `ghost`, `subtle` (tinted), `dark` (ink). Sizes `sm | md | lg`. Set `accent="cura|copia|creo"` to re-theme to a product hue; `pill` for rounded chips/CTAs; `block` for full width. Icons use Bootstrap Icons names.
