**IconButton** — icon-only action for toolbars, cart/wishlist toggles, and dismiss controls.

```jsx
<IconButton icon="cart2" label="장바구니" />
<IconButton icon="heart" variant="ghost" accent="creo" label="찜하기" />
<IconButton icon="x-lg" variant="ghost" rounded="pill" label="닫기" />
```

Always pass `label` for accessibility. Icons are Bootstrap Icons names.
