FAQ

자주 묻는 질문

AIvance는 어떤 회사인가요?
AIvance(AI + Advance)는 검증된 CodeIgniter 4(PHP) 백엔드 위에 실무형 AI 자동화를 얹어 성형·의료 광고, 이커머스, 기업 홈페이지 세 영역의 실전 소프트웨어 솔루션을 제공하는 회사입니다.
AIvance는 어떤 제품을 제공하나요?
세 가지 제품을 제공합니다. AICura는 성형·의료 광고 관리 솔루션(운영자·병원·대행사 포털, AI 일일 리포트, iOS·Android 앱), AICopia는 AI 쇼핑몰 솔루션(상품·주문·결제, 상품 설명 생성·문의 분류·리뷰 요약), AICreo는 기업 홈페이지·CMS 솔루션(테마 교체형 빠른 납품)입니다.
홈페이지 제작(AICreo) 납품 기간은 얼마나 걸리나요?
단순 홈페이지는 3~5일, 중형 사이트는 1~2주 납품을 목표로 설계되었습니다. 코어는 재사용하고 테마(디자인)만 교체하는 구조로 적용 기간을 단축합니다.
AI는 구체적으로 어떤 업무를 자동화하나요?
리포트 작성, 상품 설명 생성, 고객 문의 분류, 리뷰 요약, 이상 탐지 같은 반복 업무를 대신 처리합니다. LLM 호출은 꼭 필요한 곳에만 사용하고 나머지는 휴리스틱(규칙 기반)으로 처리해 비용과 지연을 최소화하는 비용 최적화형 설계입니다.
AIvance는 어떤 기술 스택을 사용하나요?
CodeIgniter 4(PHP 8.4+), MySQL 8, Bootstrap 5를 기반으로 하며 JWT·OAuth2 인증, Groq·Claude 등 LLM, GitHub Actions CI/CD, AWS·GCP·Azure 배포, PHPUnit 테스트, PHPStan 정적 분석을 갖춘 프로덕션 레벨 코드베이스입니다.
